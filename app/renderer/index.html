GLOSSARY
crew
a bench of specialist agents saved as .magentra/team/*.md files; the orchestrator dispatches them on tasks. One agent is a member.
backpack
a member's document knowledge — drop files on its card and they are indexed for retrieval during that member's runs.
lessons
knowledge a member earns from verified work: candidates ride on probation and are promoted only after repeated confirmation.
atlas
the whole-design map of this workspace at .magentra/ATLAS.md; /atlas builds or refreshes it.
mission
a saved directive file under .magentra/missions/ that the lab runs on demand, on a loop, or on a cron schedule (MISSIONS in the dock).
task list
the live plan of record in the TASKS rail — the current work items, ending with a verification task.
styles
.ma files that shape how the agent works; core styles are always on, optional ones toggle per session via the chips or /styles.
plan mode
a read-only mode: the agent proposes a plan you approve before it may edit anything.
deletion guard
destructive commands always ask first — in every permission mode — unless Deletions is set to ALLOW above.
session
one conversation with its own transcript, cost ledger, and task list, saved under .magentra/sessions/ (SESSIONS in the dock resumes them).
